ğŸ”§ CUSTOMIZATION
==========================================================

ADDING MORE Q&A PAIRS:
----------------------
1. Edit: dataset/apsnaturals_qa_dataset.txt
2. Add questions/answers in this format:

   Question: Your question here?
   Answer: Your answer here.
   
   (blank line between pairs)

3. Re-train: python train.py
4. Restart web server

CUSTOMIZING SUGGESTIONS:
------------------------
Edit app.py, find TOPIC_SUGGESTIONS dictionary
Add your own categories and suggested questions

CHANGING UI COLORS:
-------------------
Edit static/style.css, modify :root CSS variables:
- --primary-color: Main brand color
- --primary-light: Lighter shade
- --secondary-color: Accent color

ADJUSTING MODEL:
----------------
Edit model.py to change:
- Number of layers
- Neurons per layer
- Activation functions
Then re-train the model

==========================================================
ğŸ› TROUBLESHOOTING
==========================================================

Q: "Model file not found" error
A: Run python train.py first

Q: Port 5000 already in use
A: Change port in app.py: app.run(port=5001)

Q: NLTK stopwords error
A: Code auto-downloads on first run, just wait

Q: Suggestions not showing
A: Make sure questions_original.pkl exists
   Re-run: python train.py

Q: Low quality responses
A: Add more Q&A pairs to dataset
   Increase epochs in train.py (line with model.fit)

Q: Web page not loading
A: Check if Python firewall is blocking
   Try http://127.0.0.1:5000 instead

Q: CSS/JS not loading
A: Check templates/ and static/ folders exist
   Make sure files are in correct locations
